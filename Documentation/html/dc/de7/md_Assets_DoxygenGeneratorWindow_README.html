Doxygen© Generator Window

Help you to document easily your projects

This package is very easily to use :

  • in Unity3d Project select menu >ideMobi>Doxygen© Generator Window
  • select path to Doxygen© :
    • if Doxygen is not installed , follow instructions to download Doxygen for macOS© or Windows© at http://www.stack.nl/~dimitri/doxygen/index.html
  • enter informations for your project
  • drag and drop the folder of your scripts (scan is recursive, no panic)
  • select the folder to output the documentation
  • save your config (to restore the next time the save config)
  • generate the documentation!
  • be happy!
