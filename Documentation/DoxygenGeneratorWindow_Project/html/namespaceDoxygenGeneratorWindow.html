DGW Documentation generator is a class to generate the documentation by Doxygen automatically After reading comparison at https://en.wikipedia.org/wiki/Comparison_of_documentation_generators the challengers for c# with output HTML are ...

  • Document! X (eliminated : no macOS, no linux)
  • Doxygen
  • Natural Docs (partial PHP ...)
  • NDoc (eliminated : no read text as input!)
  • Visual Expert (eliminated : no macOS, no linux)
  • VSdocman (eliminated : no macOS, no linux) We decided to use Doxygen beacuse it's the most easy to use for a proof of concept. We will propose Natural Docs later, because the library is not really usable on macosx
